# Food Munch 🍽️

A responsive food ordering and restaurant landing page built using HTML, CSS, and Bootstrap. The website showcases menu categories, healthy food options, delivery services, payment methods, and customer engagement sections with a modern and mobile-friendly design.

## 🌐 Live Demo

Visit the website here:

https://akshayfoodmunch.ccbp.tech/

## 🚀 Features

- Responsive design for mobile, tablet, and desktop devices
- Attractive hero/banner section
- Explore menu categories
- Why Choose Us section
- Healthy food promotion section
- Delivery and payment information
- Gift voucher and customer appreciation section
- Social media follow section
- Footer with contact information

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- Font Awesome Icons
